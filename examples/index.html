Basic call user function

data-func is used to call user javascript function, the value is the user function name.
The first parameter of the user function is the html element.

Click me 
<a data-func="userFunc">Click me</a>

// In user.js
function userFunc(element) {
  console.log('[USER SCRIPT] call function !');
}

Ajax call

The value of data-link will be the url send by ajax request.

Click me 
<a data-link="/">Click me</a>

Specified event

data-event is used to define wich event is associated with the element.

Mouseover me Mouseover me 
<a data-link="/" data-event="mouseover">Mouseover me</a>
<a data-func="userFunc" data-event="mouseover">Mouseover me</a>

Ajax call with content update

With data-wrapper you can easly insert the response content to a container.
The value of data-wrapper is just a selector passed to jQuery.

Click me 
<a data-link="file.html" data-wrapper="#wrapper">Click me</a>

Ajax call with json response

You can specify wich data-type will be parsed by jQuery.

Click me 
<a data-link="file.json" data-type="json">Click me</a>

Ajax call with json response and callback function

Callback functions must avec this parameters: callbackFunction(element, data, textStatus, jqXHR, errorThrown);
Element is the div element wich start the event, data is the result of the ajax request, for textStatus and jqXHR please refer to the jquery documentation.
If success, the errorThrown will be setted to null, if not it will data wich be null.
User function must be out outside of "$(document).ready(function()"

Click me 
<a data-link="file.json" data-type="json" data-callback="userCallback">Click me</a>

function userCallback(element, data, textStatus, jqXHR, errorThrown) {
    console.log('[USER SCRIPT] Callback function !');
}

Refresh

data-refresh can be usefull to refresh a content (must be used with data-wrapper).
data-refresh value is not mandatory, but if defined you can user "refresh("refreshName")" throw javascript to do the same behavior.

Click me 
<a data-link="file.html" data-wrapper="#wrapper" data-refresh="refreshName">Click me</a>

Form

Remove or hide the submit element and add a "a" tag with data-form attribute, fill the data-form with the form name if the tag is outside the form.
You can define the "data-link" attribute instead "action" if you like, data-method and method as well.

save 
<form action="file.php" method="post" name="formName">
    <input type="text" name="name"/>
    <a data-form>save</a>
</form>
<a data-form="formName">save</a>

OR

<form name="formName">
    <input type="text" name="name"/>
    <a data-form data-link="/" data-method="post">save</a>
</form>
